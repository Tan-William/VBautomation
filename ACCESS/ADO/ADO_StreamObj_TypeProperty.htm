ADO Stream object >> Type property

Indicates the type of data contained in the Stream (binary or text). The Type property is read/write only. The default value is adTypeText. However, if binary data is initially written to a new, empty Stream, the Type will be changed to adTypeBinary.

Note:
For text data, the character set used for translation is set using the CharSet property.

StreamTypeEnum Constant

Constant Value

Description
adTypeBinary 1 Binary data
adTypeText 2 Default, text data

Syntax:

objStream.Type = StreamTypeEnum
StreamTypeEnum = objStream.Type

Example:

If (objStream.Type = adTypeText) Then
   objStream.Charset = "ascii"
End If
