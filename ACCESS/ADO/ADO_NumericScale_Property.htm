ADO NumericScale Property

The NumericScale property sets or returns a byte value that indicates the number of digits stored to the right side of the decimal point for a numeric value of a Field or Parameter object.

Object

Description of the NumericScale Property
Field The NumericScale has read/write permissions when used to create a Recordset, but it has read-only permissions when you open an existing Recordset
Parameter The NumericScale property has read/write permissions on a Parameter object

Syntax

object.NumericScale

Example

set conn=New ADODB.Connection
conn.Provider="Microsoft.Jet.OLEDB.4.0"
conn.Open "F:\MaterialMaster\Material_Master.mdb"

set rs=New ADODB.Recordset
rs.Open "SELECT * FROM tblAMSEA", conn
Debug.Print(rs.Fields(0).NumericScale)
rs.Close
conn.close
