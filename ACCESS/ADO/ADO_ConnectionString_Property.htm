ADO ConnectionString Property

The ConnectionString property sets or returns the details used to create a connection to a data source.
The Command's ConnectionString argument is identical to that used by the Connection object.

Note: You can not use both the Provider and File Name parameters.

Syntax

objconn.ConnectionString="para1=value;para2=value;etc;"

The ConnectionString property has these parameters:

Parameter

Description
Provider The provider to use for the connection
File Name A provider-specific file that contains connection information
Remote Provider The provider to use when opening a client-side connection
Remote Server A path name of the server to use when opening a client-side connection
url An absolute URL identifying a resource, such as a file or directory

Example

Sub test_ConnectionString()

Dim conn As ADODB.Connection
Dim strConn As String
Set conn = New ADODB.Connection

strConn = ConnectionString
conn.ConnectionString = "Provider=Microsoft.Jet.OLEDB.4.0;" & _
  "Data Source=F:\MaterialMaster\Material_Master.mdb;"
conn.open strConn
MsgBox "Connection to MaterialMaster database is made"

conn.close
Set conn = Nothing
End Sub
