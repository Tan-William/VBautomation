List of All RunCommand (acCommand) Constants

In Access 2003, there are 197 acCommand constants that represent menu and the toolbar items alone. Altogether, Access 2003 has constants with values 1 to 536. Access 2007 added 137 constants and their values run from 537 to 673.

A column on the version tells you when they first appeared - Access 97, Access 2000, Access XP, Access 2003 or Access 2007. Prior to Access 2007, all later versions of Access recognise the earlier commands.
