ColorChart


The ColorChart applet is a very basic tool for viewing different color combinations of text and background, useful for selecting colors to use within HTML documents. Converts both hexadecimal (used to set colors in HTML) to decimal, and decimal (used to set colors in Java) to hexadecimal. Colors can be obtained by using the red, green and blue scrollbars, or type hexadecimal or decimal code into the appropriate text box and press enter to select the color. Hex code can be entered in either upper or lower case.
